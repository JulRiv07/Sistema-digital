# 🛠 Backend - Instalación y ejecución

## 📦 Requisitos previos

- Python 3.10 o superior
- pip instalado

---

## 🚀 Pasos para ejecutar el backend

### 1️⃣ Entrar a la carpeta backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```
